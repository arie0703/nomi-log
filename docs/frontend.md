# フロントエンド実装

src/配下にフロントエンド関連ファイルを格納

## src/配下の各ディレクトリの役割

- assets/
  - styles/
    - scssファイルの格納
  - images/
    - 画像ファイルの格納
- components/
  - 再利用可能なコンポーネントファイルを定義
- utils/
  - 再利用可能な処理を格納
- hooks/
  - 再利用可能なカスタムフックを格納

## style設定

- スタイルにはscssを使う。
- コンポーネントの要素のクラス命名規則としては[component名]--[要素名]とする (例: .post-form--button)
- コンポーネントごとにscssファイルを作成する。ファイルの命名規則はチェインケースを用いる
