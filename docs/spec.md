# アプリ仕様

- お酒を飲んだ記録を日記のように記録するデスクトップアプリケーション(Vite + Tauri)
- インターネットと接続はせず、ユーザーが個人的に使用することを想定

## 機能

### 投稿機能

- 日付, コメント, 飲んだお酒の名前, 飲んだ量(ml)を投稿する
- 飲んだお酒はプルダウンでデータベースに登録済みのお酒情報から選択する。
  - プルダウンには登録済みのお酒一覧が表示される。
  - カテゴリーでプルダウンに表示される内容をソートすることが可能。
- 飲んだお酒の名前は複数選択が可能。

### 投稿表示

- 投稿された内容を一覧で表示する
- 日付, コメント・お酒の名称が表示される。

### お酒の管理機能

- お酒登録画面からお酒の情報を登録できる
- お酒の情報としては、名称, アルコール度数, カテゴリーがある
- カテゴリーにはビール・ウイスキー・リキュール・梅酒・焼酎・ワイン・ノンアルコールが初期データとしてあり、後から増えることも想定する。
- 投稿画面からもお酒を新規登録することが可能

## デザイン要件

- 各機能の画面の表示領域はウィンドウの90%とする
- タブで各画面を切り替えられる
- ダークモードを適用する
